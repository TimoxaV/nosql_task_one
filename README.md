1. Для чого використовується інструкція $unwind?
$unwind створює по окремому документу на кожне значення в масиві. Наприклад, трек із масивом виконавців artists: ["Drake", "Dr.Dre"] створить два документи: один з artists: "Drake", а інший - з artists: "Dr.Dre". Це необхідно зробити перед групуванням за окремими виконавцями.

2. Чим $stdDevPop відрізняється від $stdDevSamp?
$stdDevPop - це стандартне відхилення генеральної сукупності (ділення на N) і застосовується,коли згруповані документи є всією генеральною сукупністю (у колекції є абсолютно всі треки певного жанру).

$stdDevSamp - це вибіркове стандартне відхилення (ділення на N-1). Застосовується, коли згруповані документи є вибіркою з генеральної сукупності. Шукаючи незвичні треки за темпом ми використовували $stdDevPop, оскільки при розрахунку ми використовували всі треки, які є в колекції.
